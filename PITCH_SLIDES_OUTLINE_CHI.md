# Elite Trader Pitch Slides Outline（中文版）

## 1. 封面

- 標題：Elite Trader
- 副標：可信賴跨境 TCG 交易與體驗平台
- 一句話主張：先驗真、再定價、安全交易、最後玩起來。

## 2. 市場痛點（服務對象）

- 散戶：鑑定成本高、假貨與 re-pack 風險高、網上信任不足。
- 卡商：獲客成本高、固定成本重、地區限制強、利潤空間收窄。
- 玩家社群：活動分散、缺大型賽事體驗、線上線下對戰整合不足。

## 3. 為何現在

- TCG 市場價值與參與者正在全球化。
- 造假風險與定價失真上升速度快於現有信任基建。
- 現有渠道偏向成交效率，欠缺信任與留存飛輪。

## 4. 產品主張

- 一個平台，四大支柱：
  - 信任基建
  - 合理流通
  - 卡商效率
  - 社群成長
- 每個主張都對應可視化、可點擊的產品證據。

## 5. Demo 流程（可按制）

- 角色選擇：散戶 / 卡商 / RSA 驗卡員。
- RSA 驗卡員登入後進入 mission：
  - 接任務
  - 掃描卡牌
  - 比對序號與指紋
  - 提交判定
  - 發出驗真 badge
- 可點擊模組：
  - 跨境物流追蹤
  - HK TCG 直播中樞
  - Tournament bracket + team league API 模擬

## 6. 信任模組證據

- RSA 驗真標章與風險標記機制。
- AI 預檢降低不必要送鑑成本。
- 上架卡牌顯示託管交割狀態。
- UI 內直接展示驗真率、交割時間等 KPI。

## 7. 流通與卡商價值

- 合理價格指標 + 已驗成交參考價。
- 跨境上架 + 物流透明，降低交易摩擦。
- 透過平台需求與重複上架行為，降低 CAC。

## 8. 社群成長飛輪

- 直播中樞集中香港 TCG 流量與互動。
- 聯盟賽與 bracket 提升回訪與參與頻率。
- 社群互動轉化為上架、出價與重複交易。

## 9. Go-To-Market（切入策略）

- 先以香港作為「信任優先」切入點。
- 優先導入可信卡商與活躍玩家社群。
- 透過活動與聯盟內容建立持續流量。

## 10. KPI（前兩季）

- 驗真完成率
- 受保護交割流程下 GMV
- 賣家重複上架率
- 活動參與 -> 交易轉化率
- 直播觀看 -> 市場行為轉化率

## 11. 路線圖

- 階段一：信任與流通基礎完成。
- 階段二：物流、直播、賽事互動能力上線。
- 階段三：合作 API、地區擴張、流程自動化。

## 12. 合作邀請（Ask）

- 策略分銷夥伴與卡商導入管道合作。
- 試點活動與區域生態夥伴協作。
- 支持跨境擴張與系統整合資源。
